# VFiles 用户手册

本手册介绍 VFiles 的常见使用方式。

## 1. 文件浏览

- 进入页面后会展示当前目录文件列表。
- 点击文件夹进入子目录；通过面包屑返回上级。

## 2. 搜索

- 在底部操作栏输入关键字并执行搜索。
- 可按文件名或内容搜索（具体选项以界面为准）。

## 3. 历史（目录级版本浏览）

VFiles 支持“按某个 commit 浏览目录”：

- 切换到底部操作栏的“历史”模式。
- 会显示当前目录（或仓库根）的提交列表/当前选中 hash。
- 通过前后按钮切换 commit 后，文件列表、预览、下载等操作都会基于该 commit。
- 退出历史模式会自动回到当前版本（HEAD/worktree）。

## 4. 预览与下载

- 文本/Markdown/图片等可在页面内预览。
- 文件下载支持下载当前版本或指定 `commit` 的历史版本。
- 文件夹下载会打包为 ZIP，历史模式下会打包选中 commit 的目录快照。

## 5. 上传

- 点击上传入口选择文件，填写提交信息后上传。
- 大文件会走分块上传并支持断点续传（若中途中断，重新上传会继续缺失的分块）。

## 6. 批量操作

- 切换到批量模式后可多选文件。
- 支持批量下载/删除/移动，单选时可重命名。
